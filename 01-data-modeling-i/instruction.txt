# RUN data modeling I
1. Connect web server "nginx" and mapping port to 8080
    -> Open docker nginx
    -> docker run -p 8080:80 nginx
2. RUN cd 01-data-modeling-i to direct in folder
3. Open etl.py file and run python library to connect postgres (install package) by "pip install psycopg2" 
4. Run docker compose file (popstgres) with > $ docker compose up
5. Add another teminal to run python file that connect to postgres
6. Run create_table.py with > $ python create_table.py
7. Run etl.py with > $ python etl.py
